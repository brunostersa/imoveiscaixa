# 🏠 Dashboard de Imóveis Caixa

Aplicação Streamlit para visualização de oportunidades de imóveis com filtros, pontuação inteligente e perfis salvos por usuário.

## 🔧 Tecnologias
- Python
- Streamlit
- Firebase (Realtime Database)
- Plotly
